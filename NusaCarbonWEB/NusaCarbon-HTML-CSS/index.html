Process

How NusaCarbon works

Alur platform mengikuti proses registrasi proyek hingga retirement token, seperti yang dibutuhkan pada marketplace kredit karbon modern.

1. Project Registration

Input data proyek

Project owner mengisi identitas proyek, lokasi, metodologi, estimasi kredit, dan periode monitoring.

2. Upload & Verification

Unggah dokumen

Dokumen legal, polygon map, foto lapangan, dan laporan teknis dikirim untuk dicek oleh verifier.

3. MRV & Tokenization

Validasi data MRV

Data MRV yang lolos validasi menjadi dasar penerbitan token karbon pada blockchain ledger.

4. Marketplace Trading

Buyer membeli token

Buyer dapat memilih proyek dengan dampak, lokasi, dan harga yang sesuai target dekarbonisasi mereka.

5. Retirement Proof

Burn & certificate

Offset claim diselesaikan lewat retirement token dan sertifikat digital berbasis hash transaksi.

Double Offset Prevention

Setiap token hanya bisa retired sekali

Setelah buyer melakukan offset claim, sistem memindahkan token ke status burned/retired dan menghasilkan bukti digital yang dapat diaudit.

  • Hash transaksi blockchain tercatat
  • Status token berubah menjadi retired permanen
  • Sertifikat digital dapat diunduh
  • Riwayat offset tidak dapat digandakan
