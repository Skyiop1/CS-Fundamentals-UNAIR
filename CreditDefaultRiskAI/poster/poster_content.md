# Poster - Credit Default Risk AI

## Title

Credit Default Risk AI

## Tagline

Deep learning dashboard for credit card default risk prediction.

## System Flow

Input Raw Dataset -> Preprocessing -> Data Splitting -> SMOTE -> DNN / Hybrid PCA-DNN -> Evaluation -> Best Model -> Streamlit GUI -> SQLite History

## Visual Notes

- Use deep blue as the primary color.
- Use signal red for high-risk states.
- Use emerald green for best-model and low-risk states.
- Include the confusion matrix, model comparison chart, and dashboard screenshot.

## Notes

The final poster can be designed after training is completed so the best-model metrics and evaluation charts are available.
